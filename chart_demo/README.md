# 随机线性图展示项目

## 项目结构
```
chart_demo/
├── backend/
│   ├── app.py          # Flask 后端
│   └── requirements.txt # Python 依赖
├── frontend/
│   ├── index.html      # HTML 页面
│   ├── style.css       # 样式
│   └── app.js          # JavaScript
└── README.md
```

## 技术栈
- 后端：Python + Flask
- 前端：HTML5 + Chart.js
- 数据：随机生成线性图数据
