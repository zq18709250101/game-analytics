# 时间序列分析维度方案 v1.0

## 一、维度概述

**维度名称**: 时间序列分析  
**核心问题**: 不同日期新注册用户的长期表现如何对比？关键指标随时间如何变化？  
**分析目标**: 支持多日期对比、留存趋势、IPU趋势的综合时间序列分析

---

## 二、核心指标定义

### 2.1 基础指标

| 指标名称 | 定义 | 计算方式 | 业务意义 |
|---------|------|---------|---------|
| **首日用户数** | 当日新注册用户数 | `COUNT(DISTINCT openid) WHERE DATE(activetime) = 目标日期` | 每日新增规模 |
| **首日IPU** | 首日人均视频观看次数 | `SUM(CASE WHEN isAD=1 THEN 1 ELSE 0 END) / COUNT(DISTINCT openid) WHERE activeday = 1` | 首日变现能力 |
| **次日留存** | 第2天仍活跃的用户占比 | `COUNT(DISTINCT CASE WHEN activeday >= 2 THEN openid END) / 总用户数` | 短期留存 |
| **3日留存** | 第3天仍活跃的用户占比 | `COUNT(DISTINCT CASE WHEN activeday >= 3 THEN openid END) / 总用户数` | 短期留存 |
| **7日留存** | 第7天仍活跃的用户占比 | `COUNT(DISTINCT CASE WHEN activeday >= 7 THEN openid END) / 总用户数` | 中期留存 |
| **14日留存** | 第14天仍活跃的用户占比 | `COUNT(DISTINCT CASE WHEN activeday >= 14 THEN openid END) / 总用户数` | 中期留存 |
| **30日留存** | 第30天仍活跃的用户占比 | `COUNT(DISTINCT CASE WHEN activeday >= 30 THEN openid END) / 总用户数` | 长期留存 |
| **60日留存** | 第60天仍活跃的用户占比 | `COUNT(DISTINCT CASE WHEN activeday >= 60 THEN openid END) / 总用户数` | 长期留存 |
| **次日IPU** | 第2日人均视频观看次数 | `SUM(CASE WHEN isAD=1 AND activeday=2 THEN 1 ELSE 0 END) / COUNT(DISTINCT CASE WHEN activeday >= 2 THEN openid END)` | 次日变现 |
| **3日IPU** | 第3日人均视频观看次数 | 同上，activeday=3 | 3日变现 |
| **7日IPU** | 第7日人均视频观看次数 | 同上，activeday=7 | 7日变现 |
| **30日IPU** | 第30日人均视频观看次数 | 同上，activeday=30 | 30日变现 |
| **60日IPU** | 第60日人均视频观看次数 | 同上，activeday=60 | 60日变现 |

### 2.2 衍生指标

| 指标名称 | 定义 | 计算方式 |
|---------|------|---------|
| **留存衰减率** | 留存下降速度 | `(次日留存 - 7日留存) / 次日留存` |
| **IPU增长率** | IPU增长趋势 | `(7日IPU - 首日IPU) / 首日IPU` |
| **LTV预估** | 用户生命周期价值 | `SUM(各日IPU * eCPM) / 总用户数` |

## 三、图表规划

### 3.1 图表清单

| 序号 | 图表名称 | 类型 | X轴 | Y轴 | 系列 |
|------|---------|------|-----|-----|------|
| 1 | 多日留存对比曲线 | 折线图 | 留存天数(1-60) | 留存率(%) | 不同日期线条 |
| 2 | 多日IPU对比曲线 | 折线图 | 天数(1-60) | IPU次数 | 不同日期线条 |
| 3 | 首日用户数趋势 | 柱状图 | 日期 | 用户数 | 单日数据 |
| 4 | 首日IPU趋势 | 折线图 | 日期 | IPU | 单日数据 |
| 5 | 关键留存点对比 | 分组柱状图 | 留存节点(次日/3日/7日/14日/30日/60日) | 留存率 | 不同日期分组 |
| 6 | 关键IPU点对比 | 分组柱状图 | IPU节点(首日/次日/3日/7日/30日/60日) | IPU次数 | 不同日期分组 |
| 7 | 留存热力图 | 热力图 | 日期 | 留存天数 | 留存率颜色深浅 |
| 8 | 日期对比表格 | 表格 | 指标名称 | 各日期数值 | 多日期并列 |

### 3.2 交互功能

| 功能 | 说明 |
|------|------|
| **日期范围选择** | 选择起始日期和结束日期，显示范围内所有日期数据 |
| **日期多选** | 支持选择多个特定日期进行对比（最多5个） |
| **指标切换** | 在留存/IPU/用户数之间切换显示 |
| **时间粒度** | 支持按日/按周/按月汇总 |
